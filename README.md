# 📌 Project Overview

This project predicts motor insurance claim probability using machine learning and converts predicted risk into driver risk scores and premium recommendations.

---

# 🎯 Features

- Claim Prediction
- Logistic Regression
- Random Forest
- XGBoost
- SMOTE
- SHAP Explainability
- Driver Risk Score
- Premium Pricing Engine

---

# 📊 Machine Learning Workflow

1. Data Cleaning
2. Feature Engineering
3. Class Imbalance Handling (SMOTE)
4. Model Training
5. Model Evaluation
6. SHAP Explainability
7. Risk Scoring
8. Premium Recommendation
